

# **Draw.io Quick Launcher 隐私政策（Privacy Policy）**

**最近更新日期：2025-11-22**

欢迎使用 **Draw.io Quick Launcher**（以下简称“本扩展”）。我们非常重视用户隐私，并承诺严格保护您的个人信息。本隐私政策解释我们如何收集、使用和处理您的数据。

本扩展符合 **Chrome Web Store User Data Privacy** 的相关要求。

------

# 1. 我们收集的数据（What data we collect）

本扩展**不收集任何用户个人信息**，也不存储、不上传、不共享任何用户数据。

扩展处理的数据仅包括：

### 在用户操作时本地读取的文本

当用户：

- 选中网页中的 Draw.io / mermaid 代码并点击右键菜单
- 或点击扩展在页面中注入的“一键打开”按钮

扩展会 **在本地内存中临时读取并处理用户选中的文本内容**，以便生成 Draw.io 可识别的 URL。

### 本扩展不收集以下数据：

- 个人身份信息
- 浏览记录
- Cookies
- 登录信息
- 表单数据
- AI 对话内容（除用户主动选中或点击的图表代码外）
- 设备或系统信息

------

# 2. 我们如何使用这些数据（How we use data）

扩展仅将用户主动选中的图表代码用于以下目的：

1. **在本地压缩并编码**为 Draw.io URL 的参数
2. **打开 app.diagrams.net（Draw.io 官方网站）** 并加载代码
3. 整个处理过程仅发生在本地浏览器，无任何上传行为

扩展不会在本地硬盘、服务器或云端存储任何数据。

------

# 3. 我们如何共享数据（How we share data）

本扩展：

- **不会与任何第三方共享用户数据**
- **不会与广告商或数据经纪商共享数据**
- **不会将数据发送至开发者或外部服务器**
- **不会内置跟踪、分析或统计工具**

唯一的外部交互是：

### 用户主动点击时，将编码后的图表通过 URL Hash 方式传递给：

https://app.diagrams.net (draw.io 官方网站）

这属于浏览器直接访问网站的正常行为，不包含个人信息，也不会被其他方使用。

------

# 4. 数据存储（Data Storage）

- 扩展 **不存储任何用户数据**。
- 所有处理均在浏览器的内存环境中实时完成，关闭标签页后即被清除。
- 扩展没有服务器端组件，因此不存在服务器端存储。

------

# 5. 第三方服务（Third-Party Services）

本扩展唯一交互的网站为：

**diagrams.net / app.diagrams.net**
 用于渲染用户的图表文件。

该行为由用户主动触发，不涉及数据上传或共享。

如需了解 diagrams.net 的隐私政策，请访问其官方页面。

------

# 6. 儿童隐私（Children’s Privacy）

本扩展不面向儿童，也不会收集任何个人数据，因此不涉及 COPPA 相关风险。

------

# 7. 安全措施（Security）

由于扩展不上传、不共享、不存储用户数据，数据泄露风险极低。
 所有操作均在用户本地浏览器环境中完成。

------

# 8. 政策更新（Updates to this Policy）

我们可能会根据扩展功能更新本隐私政策。
 如有重大变动，会在 Chrome Web Store 页面进行提示。
